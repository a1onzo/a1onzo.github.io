一 前言

mysqladmin 是查看mysql状态，管理mysql常用的命令。

本文主要记录 mysqladmin命令的常用用法

二 常用用法

所有mysqladmin的参数都可以用最小唯一前缀，能提升效率。

比如 mysqladmin processlist 可以简化为 mysqladmin pr

2.1 查看mysql版本

命令：mysqladmin version

输出：

最后有一些mysql的状态信息。后面会详细解释。

2.2 root密码管理

创建密码: mysqladmin -u root password [new_password]

修改密码：mysqladmin -u root -p123456 password '1234567'

2.4 查看Mysql状态

命令：mysqladmin status / mysqladmin extended-status

mysqladmin status 显示几个指标数据

mysqladmin extended-status 显示完整的指标数据

示例：

返回结果解释

完整的官方解释：https://dev.mysql.com/doc/refman/8.0/en/server-status-variables.html

2.5 查看Mysql 配置，变量

mysqladmin variables

具体变量值参考第三章中引用的官方文档

2.6 查看mysql正在运行的线程

命令：mysqladmin processlist

2.7 启停Slave

mysqladmin start-slave

mysqladmin stop-slave

2.8 增删数据库

mysqladmin create [database_name]

mysqladmin drop [database_name]

2.9 加载授权表

mysqladmin reload

2.10 刷新表，重新打开日志

mysqladmin refresh

2.11 关闭mysql服务器

mysqladmin shutdown
